# FocusTrail Extension Icons

The extension expects three icon files:
- `icon16.png`  (16×16 px)
- `icon48.png`  (48×48 px)
- `icon128.png` (128×128 px)

## Quick generation (Node.js)

Run the script below from this folder to generate simple placeholder icons:

```bash
node generate-icons.js
```

See `generate-icons.js` in the root of this folder for details.

## Custom icon

Design a square logo, export it at 16, 48 and 128 pixels, name the files as above,
and drop them into this `icons/` directory.
