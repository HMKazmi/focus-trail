# FocusTrail Documentation Assets

This folder contains images and videos for the README documentation.

## Images

Place screenshots and images here:
- `banner.png` - Main repository banner (1200x400px recommended)
- `dashboard.png` - Dashboard screenshot showing analytics
- `mobile-app.png` - Mobile app screenshot
- `extension-ui.png` - Chrome extension screenshot
- `task-management.png` - Task management interface

## Videos

Place demo videos/GIFs here:
- `mobile-demo.gif` - Mobile app demo (800x600px, < 10MB)
- `extension-demo.gif` - Extension demo (800x600px, < 10MB)
- `full-demo.mp4` - Complete feature demonstration

## Recommended Tools

- **Screenshots**: Use OS built-in tools or ShareX (Windows)
- **GIF Creation**: ScreenToGif, LICEcap
- **Image Optimization**: TinyPNG, ImageOptim
- **Video Recording**: OBS Studio, QuickTime (Mac)

## Guidelines

- Use PNG for screenshots (better quality)
- Use GIF for short demos (< 10MB)
- Compress images before committing
- Maintain consistent aspect ratios
- Add descriptive captions in README
